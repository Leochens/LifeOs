# 你的任务
任务一：总览与基础设置优化
• 新增日历热力图
   • 在总览页集成日历热力图。
   • 后端逻辑：建立独立日志文件，实时监测并记录日记、待办（Todo）、便利贴的新增与修改操作。
   • 前端展示：基于记录的日志数据生成可视化热力图。
• 邮箱账户管理
   • 邮箱账户列表的每一项增加右键菜单，支持快速修改与编辑。



任务二：项目管理系统升级
• 项目卡片优化
   • 信息展示：卡片上需标明所属面板数量及待完成 Todo 总数。
   • 视觉精简：移除项目卡片上的优先级标识（仅保留目录属性）。
• 项目内部功能调整
   • 面板扩展：支持在默认 4 个面板基础上自定义新建面板。
   • 任务属性：面板内的 Todo 项支持设置“优先级”与“截止时间”。
   • 交互修复：修复任务项无法在不同面板间跨屏拖动的 Bug。



任务三：新增财务记账模块
• 基础架构
   • 新增“财务”页签，支持家庭成员（如爸爸、妈妈）多角色记账。
   • 自定义操作：遵循“先选人，后选项目”的添加逻辑。
• 资产更新引导
   • 采用进度图形式引导用户依次录入：流动资金、固定资产、投资理财、应收款及负债。
• 数据可视化
   • 总览看板：实时展示总资产、净资产、负债率及更新日期。
   • 对比分析：通过柱状图展示资产数值，并标注较上次记录的增减变动。
   • 多维视图：提供资产分叉图及资产月历视图。

任务 4：订阅管理
功能描述
直接录入并管理软件订阅信息（如产品 A），支持按月订阅模式的详细记录。
记录项
• 订阅金额：每月支付的具体费用
• 开始日期：订阅启动的时间节点
• 累计支出：在该产品上的历史总花费
• 续费日期：下一次扣款的具体时间
• 支付方式：记录具体的支付途径
• 应用类型：
   • 移动端 (Mobile)
   • 桌面端 (Desktop)
   • SaaS 详细信息
• 标签分类：用于自定义归类管理
展示方式
• 以列表形式进行统一展示
