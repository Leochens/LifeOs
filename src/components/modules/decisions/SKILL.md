# 重大决策技能

此技能用于通过 AI 辅助记录和管理人生中的重要决策，帮助系统化思考并追踪决策结果。

## 触发条件

- 用户输入 `/决策` 或 `/重大决策`
- 用户提及 "决策"、"选择"、"要不要"、"该不该"
- 用户询问某个重要选择的利弊分析

## 可用命令

### 创建新决策

```
/决策 创建 [决策标题] [重要性]
```

示例：
```
/决策 创建 是否换工作 高
/决策 创建 购房选择 关键
```

### 分析决策

```
/决策 分析 [决策标题]
/决策 给出建议 [决策标题]
```

### 更新决策状态

```
/决策 更新 [决策标题] 状态为 [已决定/已归档]
/决策 记录结果 [决策标题] [结果描述]
```

### 查看决策

```
/决策 列表
/决策 查看未决定
/决策 查看 [重要性] 级别的决策
```

## 重要性级别

- **低 (low)** - 影响较小，可以快速决定
- **中 (medium)** - 有一定影响，需要考虑
- **高 (high)** - 影响较大，需要仔细权衡
- **关键 (critical)** - 重大人生决策，影响深远

## 数据存储

### 决策文件路径

```
{vault}/decisions/{YYYY-MM-DD}-{slug}.md
```

### Frontmatter 格式

```yaml
---
title: 是否换工作
created: 2025-01-15
status: pending
weight: high
decided_on: ~
outcome: ~
review_date: ~
---
```

### 内容格式

```markdown
## 背景

描述这个决策的背景和上下文...

## 支持理由

- 理由一
- 理由二

## 反对理由

- 理由一
- 理由二

## 最终决定

_待定_
```

## 决策状态

1. **思考中 (pending)** - 正在收集信息和权衡
2. **已决定 (decided)** - 已做出选择
3. **已归档 (archived)** - 决策完成并已复盘

## AI 辅助功能

### AI 分析决策

使用 `runShellCommand("claude", ["-p", prompt])` 调用 Claude CLI 进行决策分析：

```
分析这个决策。标题：{title}，重要性：{weight}。

支持理由：{pros}
反对理由：{cons}

请用 3-4 句话给出建议。
```

## 数据操作指南

### 读取决策列表

使用 `listNotes("{vault}/decisions", false)` 读取所有决策文件。

### 创建新决策

使用 `writeNote` 写入新的决策文件，路径为 `{vault}/decisions/{YYYY-MM-DD}-{slug}.md`。

### 更新决策

读取现有文件，修改 frontmatter 中的 `status`、`decided_on` 或 `outcome` 字段。

## 注意事项

1. 决策标题应简洁明了，用一句话描述决策内容
2. slug 从标题自动生成，保留前 40 个字符
3. 重要性级别帮助优先处理关键决策
4. 支持和反对理由建议各列举 3-5 条
5. AI 分析功能需要安装 Claude CLI
