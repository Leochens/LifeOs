# 日记技能

此技能用于通过 AI 辅助记录个人日记和心情状态，帮助养成规律写作习惯。

## 触发条件

- 用户输入 `/日记` 或 `/写日记`
- 用户提及 "写日记"、"记录今天"、"心情"
- 用户想要回顾或总结某一天的经历

## 可用命令

### 新建日记

```
/日记 新建 [日期]
/日记 写一篇关于 [主题] 的日记
```

示例：
```
/日记 新建 2025-01-15
/日记 写一篇关于今天工作的日记
```

### 查看日记

```
/日记 查看 [日期]
/日记 今天
/日记 昨天
/日记 最近一周
```

### 搜索日记

```
/日记 搜索 [关键词]
/日记 包含 [标签] 的日记
```

### 心情分析

```
/日记 心情趋势
/日记 统计本周心情
```

## 心情选项

| 符号 | 含义 |
|------|------|
| 😴 | 疲惫/困倦 |
| 😔 | 低落/难过 |
| 😐 | 平静/一般 |
| 😊 | 愉快/开心 |
| 🔥 | 充满激情 |
| 🎯 | 有目标感 |
| 🤔 | 思考/困惑 |
| 😌 | 放松/满足 |

## 能量级别

- **high** - 精力充沛
- **medium** - 状态一般
- **low** - 疲惫乏力

## 数据存储

### 日记文件路径

```
{vault}/diary/{YYYY}/{YYYY-MM-DD}-{HHMM}.md
```

### Frontmatter 格式

```yaml
---
date: 2025-01-15
title: 今天完成了一个重要项目
mood: 😊
energy: high
tags: 工作, 成就, 感悟
---
```

### 内容格式

```markdown
# 2025-01-15

今天完成了...（自由记录内容）
```

## AI 操作指南

### 读取日记列表

使用 `listNotes("{vault}/diary/{year}", false)` 读取某年的所有日记。

### 创建新日记

使用 `writeNote` 写入新的日记文件：
- 路径：`{vault}/diary/{year}/{date}-{time}.md`
- 文件名包含日期和当前时间（HHMM 格式）

### 搜索日记

按标签搜索：读取日记文件，检查 frontmatter 中的 `tags` 字段。
按日期搜索：使用日期直接定位文件。

## 标签建议

- **活动类别**：工作、学习、运动、社交、娱乐
- **情感状态**：开心、焦虑、平静、激动
- **地点**：家、公司、咖啡厅、旅行
- **人物**：家人、朋友、同事

## 注意事项

1. 同一天可以写多篇日记
2. 每篇日记建议记录心情和能量状态
3. 标签使用逗号分隔
4. 标题可选，留空则显示 "(无标题)"
5. 日记按年份分目录存储
