# 仪表盘技能

此技能用于查看 LifeOS 的全局概览，包括任务、项目、日记、习惯等汇总信息。

## 触发条件

- 用户输入 `/仪表盘` 或 `/dashboard`
- 用户询问 "今天怎么样"、"概览"、"汇总"
- 用户需要查看整体状态

## 可用命令

### 查看概览

```
/仪表盘
/仪表盘 今天
/仪表盘 概览
```

### 查看统计

```
/仪表盘 统计
/仪表盘 任务
/仪表盘 项目
/仪表盘 习惯
```

示例：
```
/仪表盘 今天完成任务数
/仪表盘 活跃项目列表
/仪表盘 习惯完成情况
```

## 仪表盘组件

### 统计卡片

| 卡片 | 显示内容 | 点击跳转 |
|------|----------|----------|
| 待完成任务 | 未完成任务数 + 进度条 | 每日任务 |
| 进行项目 | 活跃项目数 | 看板 |
| 日记条数 | 总日记数 | 日记 |
| 待决策 | 待决策数量 | 决策 |

### 今日任务预览

- 显示前 8 条任务
- 显示完成状态
- 显示时间和标签
- 点击查看全部任务

### 活跃项目

- 显示前 3 个活跃项目
- 显示优先级和进度
- 显示更新时间
- 点击进入看板

### 今日习惯

- 显示所有习惯列表
- 点击切换完成状态
- 显示完成进度

### 活跃度热力图

- 显示最近 365 天的活动
- 颜色深浅表示活跃程度
- 统计日记、任务、项目、笔记

### 最近日记

- 显示最近 3 条日记
- 显示心情和日期
- 显示标签
- 点击进入日记

## 数据来源

仪表盘汇总来自各模块的数据：

| 数据 | 来源 |
|------|------|
| 今日任务 | `daily/tasks/{YYYY-MM-DD}.md` |
| 项目 | `projects/*.md` |
| 日记 | `diary/*.md` |
| 习惯 | `daily/habits/habits.yaml` |
| 决策 | `decisions/*.md` |
| 便签 | `sticky/*.md` |

## 活跃度计算

### 热力图等级

```
等级 0: 无活动
等级 1: 1-2 项活动
等级 2: 3-4 项活动
等级 3: 5-6 项活动
等级 4: 7+ 项活动
```

### 活动类型

- 日记条目
- 任务完成
- 项目更新
- 便签创建

## 问候语

根据时间显示不同问候：

- 6:00 前 - "夜深了，注意休息"
- 6:00-12:00 - "早上好！"
- 12:00-18:00 - "下午好！"
- 18:00+ - "晚上好！"

## AI 操作指南

### 汇总数据

从各模块读取数据并计算统计信息：

1. 读取今日任务文件
2. 读取项目列表
3. 读取日记列表
4. 读取习惯配置
5. 计算统计数据

### 生成报告

```
/仪表盘 生成周报
/仪表盘 生成月报
```

### 配置文件

```
{vault}/.lifeos/dashboard.yaml
```

## 注意事项

1. 仪表盘数据实时更新
2. 点击卡片可快速跳转到对应模块
3. 热力图统计范围是最近 365 天
4. 习惯打卡状态会立即保存
